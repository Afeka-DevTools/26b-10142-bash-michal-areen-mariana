## הוראות התקנה והגדרה ראשונית

על מנת להשתמש בסקריפטים שבמאגר זה, יש לבצע את השלבים הבאים:

1. **שכפול הפרויקט:**
   פתחו את הטרמינל והריצו את הפקודה הבאה כדי לשכפל את הפרויקט אליכם למחשב:
   ```bash
   git clone <your_repository_address>
   ```

2. **מעבר לתיקיית הסקריפטים:**
   כל קבצי הסקריפט ממוקמים בתיקייה הייעודית בשם `scripts` בתיקיית השורש של הפרויקט. כנסו אליה:
   ```bash
   cd <project_folder>/scripts
   ```

3. **מתן הרשאות ריצה:**
   לפני הרצת סקריפט כלשהו, חובה לתת לו הרשאת ריצה (Executable). ניתן לתת הרשאה לכל הסקריפטים יחד באמצעות הפקודה:
   ```bash
   chmod +x *.sh
   ```
   *הערה: כל הסקריפטים מתחילים בשורת ה-Shebang (`#!/bin/bash`) כנדרש. חלק מהסקריפטים בודקים ומתקינים אוטומטית כלים חיצוניים במידת הצורך.*

## פירוט הסקריפטים והוראות הרצה

### הסט של מיכל (סט 1)

* **גיבוי תכולה של תיקיה לקובץ tar.gz**
  * **שם הקובץ:** `backup_tar.sh`
  * **תיאור:** הסקריפט מקבל נתיב לתיקייה ויוצר קובץ ארכיון מכווץ שלה.
  * **הרצה:** `./backup_tar.sh <path_to_directory>`

* **ספירת שורות, מילים ותווים בכל קובץ בתיקיה**
  * **שם הקובץ:** `count_files_content.sh`
  * **תיאור:** הסקריפט עובר על כל הקבצים בתיקייה שסופקה ומדפיס את הסטטיסטיקות עבור כל אחד.
  * **הרצה:** `./count_files_content.sh <path_to_directory>`

* **שליפת מזג האוויר המעודכן לעיר מסוימת**
  * **שם הקובץ:** `weather.sh`
  * **תיאור:** הסקריפט מבקש מהמשתמש להזין שם של עיר ומציג את מזג האוויר הנוכחי (דורש `curl`).
  * **הרצה:** `./weather.sh` (הסקריפט יבקש קלט במהלך הריצה).

* **סיכום מספר הקבצים, התיקיות והלינקים בתיקייה**
  * **שם הקובץ:** `summary_dir.sh`
  * **תיאור:** הסקריפט מנתח את התיקייה הנתונה ומדפיס סיכום כמותי של סוגי הפריטים בתוכה.
  * **הרצה:** `./summary_dir.sh <path_to_directory>`

* **מידע על המשתמש הנוכחי**
  * **שם הקובץ:** `user_info.sh`
  * **תיאור:** מדפיס למסך את שם המשתמש, תיקיית הבית, הקבוצות אליהן הוא שייך, וה-Shell המוגדר לו.
  * **הרצה:** `./user_info.sh`

### הסט של ארין (סט 2)

* **בדיקת חיבוריות לרשת האינטרנט והדפסת log**
  * **שם הקובץ:** `check_internet.sh`
  * **תיאור:** מבצע פינג לשרת חיצוני, מציג את התוצאה ושומר אותה לקובץ לוג מקומי.
  * **הרצה:** `./check_internet.sh`

* **מחיקת קבצים הישנים יותר מ-X ימים**
  * **שם הקובץ:** `delete_old_files.sh`
  * **תיאור:** הסקריפט מקבל נתיב לתיקייה ומספר ימים, ומוחק קבצים שגילם עולה על המספר שצוין.
  * **הרצה:** `./delete_old_files.sh <path_to_directory> <number_of_days>`

* **סריקת פורטים פתוחים בכתובת IP**
  * **שם הקובץ:** `scan_ports.sh`
  * **תיאור:** הסקריפט מקבל כתובת IP וסורק אילו פורטים פתוחים בה
