פתחו את הטרמינל והריצו את הפקודה הבאה כדי לשכפל את הפרויקט אליכם למחשב:  Bashgit clone <your_repository_address>
מעבר לתיקיית הסקריפטים:
כל קבצי הסקריפט ממוקמים בתיקייה הייעודית בשם scripts/ בתיקיית השורש של הפרויקט. כנסו אליה:  Bashcd <project_folder>/scripts
מתן הרשאות ריצה:
לפני הרצת סקריפט כלשהו, חובה לתת לו הרשאת ריצה (Executable). ניתן לתת הרשאה לכל הסקריפטים יחד באמצעות הפקודה:  Bashchmod +x *.sh
הערה: כל הסקריפטים מתחילים בשורת ה-Shebang (#!/bin/bash) כנדרש. חלק מהסקריפטים בודקים ומתקינים אוטומטית כלים חיצוניים במידת הצורך (כמו curl או nmap).  פירוט הסקריפטים והוראות הרצה   הסט של מיכל (סט
